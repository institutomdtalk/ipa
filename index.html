Vowel Sound Map

O IPA representa sons, não letras. Nesta página, cada som vocálico é conectado a uma imagem-chave para criar memória visual, auditiva e articulatória. A lógica de estudo é simples: imagem → palavra-chave → som → novas palavras.

Exemplo: CAT representa o som /æ/. Ao ver a imagem do gato, você lembra da palavra cat, reproduz o som /æ/ e transfere esse padrão para palavras como black, have, hat e catch.

Como estudar

Treino guiado por associação

Use cada card como um pequeno laboratório: primeiro reconheça a imagem, depois pronuncie a palavra-chave, isole o som e transfira esse som para o banco de palavras.

01

Veja a imagem

A imagem ativa uma palavra conhecida, como bird, fish, tree ou phone.

02

Pronuncie devagar

Comece pelo som, não pela escrita. Repita a palavra-chave e perceba a vogal central do card.

03

Transfira o padrão

Leve a mesma qualidade vocálica para outras palavras, como bird → nurse → work → world.

04

Marque progresso

Clique em Praticar quando o som estiver mais estável, natural e fácil de reproduzir.
